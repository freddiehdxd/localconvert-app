# Screenshots Needed for README

Take these screenshots of LocalConvert running and save them in this folder.

- Full app window showing the main interface
- Include some sample files in the queue
- Resolution: 1280x800 or higher
- This is the first thing users see!

- Show files being dragged onto the drop zone
- Or show the empty drop zone state

### 3. `batch-conversion.png`

### 4. `video-trimmer.png`

## Tips for Great Screenshots

1. **Use a clean state** - Close other apps, clean desktop
2. **Consistent size** - 1280x800 recommended
3. **Add sample files** - Use diverse file types
4. **Show features in action** - Don't just show empty states
5. **PNG format** - Best quality for UI screenshots
6. **Optimize size** - Keep under 500KB each if possible

## Image Dimensions

| Screenshot | Recommended Size |
|------------|------------------|
| Hero (main-interface) | 1280x800 |
| Feature screenshots | 1000x600 |
| Detail shots | 800x500 |

## After Taking Screenshots
